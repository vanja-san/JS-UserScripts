# Project Summary

## Overall Goal
Разделение основного скрипта NRL.user.js на два файла: один с основной логикой скрипта, а другой с отдельными строками перевода для упрощения поддержки и обновления переводов.

## Key Knowledge
- Используется формат пользовательского скрипта (UserScript) с поддержкой Tampermonkey, ScriptCat и Violentmonkey
- Основной скрипт NRL.user.js содержит массивную секцию с переводами в объекте DICTIONARIES
- Необходимо создать отдельный файл с переводами и подключить его к основному скрипту
- Пользовательские скрипты используют метаданные @require для подключения внешних файлов
- После изменений необходимо проводить сборку проекта для выявления ошибок
- Используется глобальная переменная window.NRL_TRANSLATIONS для передачи переводов между файлами

## Recent Actions
- Создана копия NRL.user.js с суффиксом -dev (NRL-dev.user.js) для разработки
- Проанализирована структура основного файла и выделена секция с переводами
- Создан отдельный файл translations.js с полным содержимым объекта DICTIONARIES
- В NRL-dev.user.js добавлена директива `@require ./translations.js` в метаданные
- В NRL-dev.user.js заменена локальная константа DICTIONARIES на `window.NRL_TRANSLATIONS || { main: {} }`
