### ℹ️ Description
Adds a 'Copy all tags' button to the thumbnail. Copy all of a tooltip tags instantly!

### ⭐ Current features:
 - Adds a one-click copy button to image Safebooru
 - Copies all tags to clipboard
 - Adds a Settings menu in Tampermonkey
